
# 贝可提醒机

## 需求

个人的一些事件记录，并允许定时提醒或做一些处理，常见场景：

- 生日提醒，每年都要提醒，并可提前5天、3天、1天各提醒一次；
- 临时事件，如10天后去医院复查，只提醒一次；

提醒方案：

- 短信、邮件、钉钉群通知等；
- 时间到，自动调用某个URL。

其它：

- SaaS机制，事件按用户ID隔离；
- 事件可以通知多人，或调用多个URLL。
